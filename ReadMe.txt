# Password Strength Checker

This is a Python-based command-line tool designed to assess the strength of user-entered passwords.  
It analyzes multiple factors such as length, character diversity, and repetition to provide a security rating and suggestions for improvement.

---

## Features

- Evaluates password length
- Checks for the presence of:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Detects repeated or overly simplistic patterns
- Returns a verdict: Weak, Moderate, or Strong
- Offers suggestions for improving weak or moderate passwords
